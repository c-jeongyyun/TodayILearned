## Trouble Shooting

### vscode prettier

1. yarn berry 사용 시 prettier 적용 안되는 문제 해결

   **환경 및 현상**

   현재 yarn berry 를 사용중이다. prettier이 적용되지 않는 문제가 발생하였다.

   **원인**

   **해결**
   설정 (`cmd` + `,` ) -> prettier 검색 -> prettier path 에 `.yarn/sdks/prettier`써주기
